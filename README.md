# ESP32 / ESP8266 学习开发仓库
个人嵌入式开发学习仓库，用于存储 ESP32、ESP8266 系列开发过程中的练习代码与功能案例，记录常用硬件模块的标准化程序，方便学习回顾与代码复用。

## 仓库用途
1. **学习记录**：保存开发学习过程中的测试代码、练习程序
2. **模块归档**：整理 WiFi、蓝牙、网络通信等通用功能模块
3. **代码复用**：沉淀可直接使用的标准化程序，提升后续开发效率

## 包含功能模块
- WiFi 连接与网络通信
- 蓝牙 BLE 基础功能
- ESP8266 + 巴法云 TCP 远程控制
- 通用外设驱动（LED、继电器等）

## 文件夹说明
- `ESP8266-Bemfa-TCP-Demo`：ESP8266 巴法云 TCP 远程控制案例
- `esp32s3-wifi-ble`：ESP32 WiFi + 蓝牙功能学习代码
- 其他文件夹：对应不同学习阶段的开发练习代码

## 使用说明
本仓库为个人学习使用，所有代码均为开发实测可用版本，可直接参考、编译、上传至对应硬件设备。
